# Thesis Pre-Defense Presentation
## Token Stacking Meta Learner — Interactive Slide Deck

### How to use

1. **Download all 3 files** (`index.html`, `styles.css`, `script.js`) into the same folder.
2. **Double-click `index.html`** — it opens in any modern browser. No server needed.
3. For best experience: press **F** to enter fullscreen mode.

### Keyboard controls

| Key | Action |
|---|---|
| `→` or `Space` or `PageDown` | Next slide |
| `←` or `PageUp` | Previous slide |
| `1` – `9` | Jump to slide by number |
| `Home` | Go to first slide |
| `End` | Go to last slide |
| `F` | Toggle fullscreen |
| `ESC` | Exit fullscreen |

You can also **swipe left/right on touch devices** or **scroll with mousewheel**.

### Tips for pre-defense day

- Open in **fullscreen** (`F`) before judges arrive.
- The progress bar at top shows your position.
- Slide counter (top-right) helps you pace yourself — aim for ~1.5 minutes per slide.
- If a judge asks to "go back to the methodology," press `7` instantly.
- Designed for 1080p projectors — looks great on any modern display.

### Aesthetic

Classical thesis pre-defense style — clean ivory background, deep navy typography (Cormorant Garamond serif), with burgundy accents for emphasis. No corporate gradients. No generic AI look.

Good luck! 🎓
